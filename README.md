## Getting started

1. Clone sample apps

   ``` shell
   $ git clone https://github.com/cloud-hackathon/ruby-apps.git
   ```

2. Move to `ruboty-bot` directory
3. Run application

   ``` shell
   SLACK_TOKEN=xoxb-abcdefghijklmnopqrstuvwxyz0123456789 bundle exec ruboty --load app.rb
   ```

4. Access Slack
